## Getting started locally

Install the dependencies:

```bash
yarn install
# or
npm install
```

Run the the development server:

```bash
yarn start
# or
npm run start
```

Compile, minify, and/or copy files for production. The compiled files will be placed in the `/dist` directory.

```bash
yarn build
# or
npm run build
```
